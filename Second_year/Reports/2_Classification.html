Суть метода k-ближайших соседей¶

Метод k ближайших соседей (англ. k-nearest neighbor algorithm) - метод автоматической классификации объектов. Основным принципом метода ближайших соседей является то, что объект присваивается тому классу, который является наиболее распространённым среди соседей данного элемента.

Соседи берутся исходя из множества объектов, классы которых уже известны, и, исходя из ключевого для данного метода значения k высчитывается, какой класс наиболее многочислен среди них.

Суть наивного байесовского алгоритма¶

Наивный байесовский алгоритм — это вероятностный алгоритм машинного обучения, основанный на применении теоремы Байеса и используемый в самых разных задачах классификации.

$$ P(A|B)=\frac{P(B|A)*P(A)}{P(B)}\\ $$
  • P(A|B) - вероятность гипотезы А при наступлении события В, т. е. вероятность А, когда В уже верно;
  • P(B|A) - вероятность наступления события В, если гипотеза А верна;
  • P(A) - вероятность того, что гипотеза А верна;
  • P(B) - вероятность наступления события В;

SVM¶

Метод опорных векторов (Support Vector Machine - SVM) относится к группе граничных методов. Она определяет классы при помощи границ областей.

При помощи данного метода решаются задачи бинарной классификации.

В основе метода лежит понятие плоскостей решений.

Плоскость (plane) решения разделяет объекты с разной классовой принадлежностью.

Реализация метода kNN¶

In [40]:
def methodknn(traindata, testdata, k, numberofclasses):
    def distance(x,y):
        return math.sqrt((x[0] - y[0])**2 + (x[1] - y[1])**2)
    ans = []
    for testpoint in testdata:
        testdist = [[distance(testpoint,traindata[i]), int(traindata[i][2])] for i in range(len(traindata))]
        stat = [0 for i in range(numberofclasses)]
        for d in sorted(testdist)[0:k]:
            stat[d[1]] += 1
        ans.append(sorted(zip(stat, range(numberofclasses)),reverse=True)[0][1])
    return ans
